# Color Update Reference

This file documents the color mappings for reference when updating files to use the theme.

## Color Mappings:
- `#4A7C9E` → `colors.primary`
- `#f5f5f5` → `colors.background`
- `#fff` / `'#fff'` → `colors.backgroundLight` or `colors.cardBackground` (depending on context)
- `#333` → `colors.text`
- `#666` → `colors.textSecondary`
- `#999` → `colors.textTertiary`
- `#ddd` → `colors.border`
- `#e0e0e0` → `colors.borderLight`
- `#f0f0f0` → `colors.borderMedium`
- `#ff5252` → `colors.error`
- `#d32f2f` → `colors.errorText`
- `#1a1a1a` → `colors.backgroundDark`
- `#2a2a2a` → `colors.backgroundDarker`
- `#3a3a3a` → `colors.backgroundMedium`

## Import Statement to Add:
```javascript
import { colors } from '../theme/colors';
```
or for screens in screens directory:
```javascript
import { colors } from '../theme/colors';
```
